# Security Policy

WP HookLens is a static scanner and should not execute scanned PHP code.

## Reporting a vulnerability

Please do not open a public issue for security-sensitive reports. Email the maintainer or use GitHub private vulnerability reporting when available.

Include:

- A description of the issue
- Steps to reproduce
- Affected versions or commits
- Any suggested mitigation

## Scope

Security reports may include unsafe file handling, report generation issues, dependency vulnerabilities, or behavior that could execute scanned project code.
